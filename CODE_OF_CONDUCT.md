# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Participants should communicate respectfully, assume good intent when possible,
and focus disagreement on facts, design, safety and user impact.

Unacceptable behavior includes harassment, threats, discrimination, publishing
private information, or using the project to expose real passenger, driver,
vehicle or safety-incident data.

## Enforcement

Maintainers may remove comments, reject contributions or restrict
participation when behavior puts contributors, users, operators, passengers
or transit staff at risk.

Report conduct concerns privately to the maintainers through the
cloud-itonami organization.
